# Ten Fish Labs

Marketing site for Ten Fish Labs — operational review software. Vite + React + Tailwind.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Structure

- `src/components/` — Nav, Footer, primitives, visuals
- `src/pages/` — Home, Products, How, About, Contact
- `public/assets/` — product screenshots
- `vercel.json` — SPA rewrites
